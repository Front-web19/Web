Fundamentals of HTML

Welcome to the fundamentals of HTML! HTML (HyperText Markup Language) is the backbone of every website. Let's explore its core concepts.

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to structure content on the web.

2. Basic HTML Structure

Every HTML page has a standard structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

3. Common HTML Tags

Here are some essential HTML tags:

  • <h1> - <h6>: Headings
  • <p>: Paragraphs
  • <a href="">: Links
  • <img src="">: Images
  • <ul>, <ol>: Lists
  • <table>: Tables

4. Adding Links & Images

Links (<a>) and images (<img>) make web pages more interactive.

<a href="https://www.example.com">Visit Example</a>
<img src="image.jpg" alt="A sample image">

5. Creating Forms

Forms collect user data. A simple form example:

<form>
  <label for="name">Name:</label>
  <input type="text" id="name">
  <input type="submit" value="Submit">
</form>

6. Semantic HTML

Semantic HTML improves readability and SEO.

<header>Website Header</header>
<nav>Navigation Menu</nav>
<section>Main Content</section>
<footer>Website Footer</footer>

7. Conclusion

HTML is the foundation of web development. Mastering it is the first step to becoming a web developer!
