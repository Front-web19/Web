What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages and web applications. It's the foundation of all websites, providing the structure and content of a webpage, which is then styled with CSS and made interactive with JavaScript.

Basics of HTML

HTML is a markup language using tags to structure content.

Tags are written as:

<tagname>Content</tagname>

HTML Structure

A basic HTML document looks like this:


<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph of text.</p>
</body>
</html>

Common HTML Tags

Tag Description Example Usage
<h1> - <h6> Headings from largest (h1) to smallest (h6) <h1>Title</h1>
<p> Paragraph <p>This is a paragraph.</p>
<a> Hyperlink <a href="https://example.com">Visit</a>
<img> Image <img src="image.jpg" alt="Description">
<div> Division or Block Container <div>Content Here</div>
<span> Inline Container <span>Inline text</span>
<ul>, <ol>, <li> Lists (Unordered & Ordered) <ul><li>Item 1</li><li>Item 2</li></ul>
<table>, <tr>, <td>, <th> Table Structure <table><tr><th>Header</th></tr><tr><td>Data</td></tr></table>
<form> Form for User Input <form><input type="text"><button>Submit</button></form>
<input> Input Field <input type="text" placeholder="Enter text">
<button> Clickable Button <button>Click Me</button>
<textarea> Multi-line Text Input <textarea>Type here...</textarea>
<select>, <option> Dropdown Menu <select><option>Option 1</option></select>
<iframe> Embedded Content <iframe src="page.html"></iframe>
<audio> Audio Player <audio controls src="audio.mp3"></audio>
<video> Video Player <video controls src="video.mp4"></video>
<nav> Navigation Links <nav><a href="#home">Home</a></nav>
<section> Section of Content <section>Content here...</section>
<article> Article Content <article>Article content here...</article>
<aside> Sidebar Content <aside>Related links here...</aside>
<footer> Footer Section <footer>Footer information here...</footer>

HTML Forms

HTML forms are used to collect user input. Here’s an example:


<form action="submit.php" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <button type="submit">Submit</button>
</form>

Comments in HTML

Comments are added to HTML for readability and notes:

<!-- This is a comment -->

Best Practices

  • Use Semantic Tags: Like <header>, <nav>, <section>, etc.
  • Always Declare <!DOCTYPE html>.
  • Use alt Attribute for Images for better accessibility.
  • Keep Code Indented and Well-Formatted for readability.
