Introduction to JavaScript

JavaScript is a versatile programming language that allows you to make web pages interactive. It can update and change both HTML and CSS, and is widely used for various applications.

Variables

Variables are used to store data. In JavaScript, you can declare variables using var, let, or const.

let name = "Alice";
const age = 25;
var city = "Lagos";

Data Types

Type Description Example
String Textual data "Hello, World!"
Number Numeric values 42, 3.14
Boolean True or False true, false
Object Collection of key-value pairs { name: "Alice", age: 25 }

Functions

Functions are reusable blocks of code. You can define them using function keyword or arrow functions.

function greet() {
    console.log("Hello, World!");
}
greet();

DOM Manipulation

JavaScript allows you to interact with and modify HTML content dynamically.

document.getElementById("demo").innerHTML = "Hello, JavaScript!";

Events

JavaScript can respond to various events such as clicks, hover, keypress, etc.

<button onclick="alert('Button Clicked!')">Click Me</button>

Loops

Loops are used to perform repetitive tasks.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Arrays

Arrays are used to store multiple values in a single variable.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]);

Objects

Objects are collections of related data and functions.

let car = {
    brand: "Toyota",
    model: "Camry",
    year: 2022
};
console.log(car.brand);
