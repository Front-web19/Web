Introduction to Bootstrap

Bootstrap is a popular CSS framework for developing responsive and mobile-first websites. It includes HTML, CSS, and JavaScript components for creating modern web designs quickly.

To use Bootstrap, add the following link to your HTML file's <head>:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.0/css/bootstrap.min.css">

Buttons

Bootstrap offers various styles of buttons. Here are some examples:

<button class="btn btn-primary">Primary</button>

Grid System

Bootstrap's grid system uses rows and columns to arrange content. It's fully responsive.

Column 1
Column 2
Column 3

<div class="row">
    <div class="col-md-4">Column 1</div>
    <div class="col-md-4">Column 2</div>
    <div class="col-md-4">Column 3</div>
</div>

Cards

Bootstrap cards provide a flexible content container with options for headers, footers, images, etc.

Card Title

This is a simple card example using Bootstrap.

Go somewhere 

<div class="card" style="width: 18rem;">
    <img src="image.jpg" class="card-img-top" alt="Sample Image">
    <div class="card-body">
        <h5 class="card-title">Card Title</h5>
        <p class="card-text">This is a simple card example using Bootstrap.</p>
        <a href="#" class="btn btn-primary">Go somewhere</a>
    </div>
</div>

Tables

Bootstrap provides various table styles for creating attractive tables.

Name Age City
John 25 Lagos
Bola 22 Abuja 

<table class="table table-bordered">
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>City</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>John</td>
            <td>25</td>
            <td>Lagos</td>
        </tr>
    </tbody>
</table>
