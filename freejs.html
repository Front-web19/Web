Get Started with JavaScript

JavaScript is a powerful programming language that allows you to add interactivity to web pages.

1. What is JavaScript?

JavaScript (JS) is used to create dynamic and interactive websites. It allows you to update content, control multimedia, and respond to user actions.

2. Adding JavaScript to HTML

You can add JavaScript in three ways:

  • Inline JS - inside an HTML tag
  • Internal JS - inside a <script> tag
  • External JS - in a separate .js file
<script>
  alert("Hello, JavaScript!");
</script>

3. Variables in JavaScript

Variables store data in JavaScript.

let name = "John";
let age = 25;
console.log(name + " is " + age + " years old.");

4. Functions in JavaScript

Functions allow you to organize code into reusable blocks.

function greet() {
  alert("Welcome to JavaScript!");
}
greet();

5. Event Listeners

JavaScript can respond to user actions like button clicks.

document.getElementById("btn").addEventListener("click", function() {
  alert("Button clicked!");
});

6. DOM Manipulation

JavaScript allows you to modify HTML elements dynamically.

document.getElementById("demo").innerHTML = "Hello, JavaScript!";

Click the button to change this text.

7. Conditional Statements

Conditional statements allow JavaScript to make decisions.

let age = 18;
if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

8. Loops in JavaScript

Loops repeat a block of code multiple times.

for (let i = 1; i <= 5; i++) {
  console.log("Number " + i);
}

9. Conclusion

JavaScript makes websites interactive and dynamic. Start practicing and build amazing projects!
