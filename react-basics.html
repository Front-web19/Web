What is React?

React is a JavaScript library developed by Facebook for building user interfaces. It allows developers to build reusable UI components and offers high performance with the use of a Virtual DOM.

Why Use React?

  • Component-Based Architecture: Reusable UI components.
  • Virtual DOM: Efficient updating and rendering of UI.
  • Declarative UI: Easy to debug and understand.
  • Hooks: Manage state and side effects in functional components.
  • Large Ecosystem: Huge community and powerful libraries.

Handling Forms

Handling forms in React is straightforward using state management.


import React, { useState } from 'react';

function FormExample() {
const [name, setName] = useState("");

const handleSubmit = (e) => {
    e.preventDefault();
    alert(Hello, ${name});
}

return (
    <form onSubmit={handleSubmit}>
        <input 
            type="text" 
            value={name} 
            onChange={(e) => setName(e.target.value)} 
            placeholder="Enter your name" 
        />
        <button type="submit">Submit</button>
    </form>
);
}

export default FormExample;

Context API

Provides a way to pass data through the component tree without having to pass props down manually at every level.


import React, { createContext, useContext } from 'react';

const UserContext = createContext();

function App() {
return (
    <UserContext.Provider value="Ade">
        <Welcome />
    </UserContext.Provider>
);
}

function Welcome() {
const user = useContext(UserContext);
return <h1>Hello, {user}!</h1>;
}

export default App;

name: 'counter',

});

const store = configureStore({
reducer: counterSlice.reducer
});

Advanced React Concepts

Hooks

Hooks allow functional components to have state and lifecycle features.


import React, { useState, useEffect } from 'react';

function Counter() {
const [count, setCount] = useState(0);

useEffect(() => {
    document.title = You clicked ${count} times;
}, [count]);

return (
    <div>
        <p>Count: {count}</p>
        <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
);
}

Error Boundaries


import React, { Component } from 'react';

class ErrorBoundary extends Component {
constructor(props) {
    super(props);
    this.state = { hasError: false };
}

static getDerivedStateFromError() {
    return { hasError: true };
}

render() {
    if (this.state.hasError) {
        return <h1>Something went wrong.</h1>;
    }
    return this.props.children;
}
}

Testing React Apps

React applications can be tested using tools like Jest and React Testing Library.


import { render, screen } from '@testing-library/react';
import App from './App';

test('renders hello world text', () => {
render(<App />);
const linkElement = screen.getByText(/Hello World/i);
expect(linkElement).toBeInTheDocument();
});
