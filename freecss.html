Discover How to Style Your Page with CSS

CSS (Cascading Style Sheets) allows you to create visually stunning websites. Let's explore its fundamentals.

1. What is CSS?

CSS is used to control the style and layout of web pages. It allows you to change colors, fonts, spacing, and animations.

2. Applying CSS to HTML

You can apply CSS in three ways:

  • Inline CSS - inside an HTML tag
  • Internal CSS - inside a <style> tag in the HTML file
  • External CSS - in a separate .css file
<style>
  body { background-color: lightblue; }
</style>

3. Changing Text Style

Use CSS to change fonts, sizes, and colors.

body {
  font-family: Arial, sans-serif;
  color: white;
  font-size: 18px;
}

4. Adding Backgrounds & Colors

CSS allows you to add colors and backgrounds.

body {
  background-color: black;
  color: white;
}

5. Styling Buttons

Make buttons look attractive.

button {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
}

6. Creating Animations

CSS can be used to create cool animations.

@keyframes glow {
  from { text-shadow: 0 0 10px red; }
  to { text-shadow: 0 0 20px yellow; }
}

7. Responsive Design

Use CSS media queries to make websites fit all screen sizes.

@media (max-width: 600px) {
  body { font-size: 16px; }
}

8. Conclusion

CSS is essential for web design. Mastering it will help you create beautiful and responsive websites.
