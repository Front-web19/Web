R is a programming language and environment specifically designed for statistical computing and data analysis. It is widely used by data scientists, statisticians, and researchers for processing large datasets, performing complex statistical computations, and visualizing data through graphs and plots.

R provides built-in support for mathematical and statistical functions, making it a powerful tool for data manipulation. It also has a vast ecosystem of packages that extend its capabilities. 

# Checking the R version installed
R.version.string

Installing and Setting Up R

To start using R, you need to install the R software. You can download it from CRAN (The Comprehensive R Archive Network). After installation, it is recommended to install RStudio, an integrated development environment (IDE) that provides a user-friendly interface for writing and running R code.

Steps to Install R:

  • Go to the official R website and download the latest version for your operating system.
  • Run the installer and follow the setup instructions.
  • Install RStudio (optional but recommended) for better code management.
  • Open R or RStudio and start coding!

Understanding Data Types in R

R supports various data types, which are essential for handling and manipulating data. Below are the primary data types in R:

  • Numeric: Represents numbers (e.g., 5.6, -3.2).
  • Integer: Whole numbers, specified with L (e.g., 10L).
  • Character: Text or string data (e.g., "Hello, World!").
  • Logical: Boolean values (TRUE or FALSE).
  • Complex: Complex numbers (e.g., 3+2i).
# Checking data types in R
class(10.5)  # Numeric
class(10L)   # Integer
class("Hello")  # Character
class(TRUE)  # Logical

Variables and Operators

In R, variables are used to store values, which can be referenced and manipulated throughout a program. Variables can hold numbers, text, or more complex data structures such as lists or data frames.

Below are different ways to assign values to variables in R:

# Assigning values to variables
x <- 5
y = "Hello R"
z <<- TRUE

# Printing variables
print(x)
print(y)
print(z)

Operators in R

R supports several types of operators, including:

  • Arithmetic operators: +, -, *, /, ^
  • Relational operators: ==, !=, >, <, >=, <=
  • Logical operators: &&, ||, !
# Example of operators
a <- 10
b <- 5
sum <- a + b  # Addition
product <- a * b  # Multiplication
is_equal <- a == b  # Logical comparison

print(sum)
print(product)
print(is_equal)

Control Structures in R

R allows you to control the flow of execution using conditional statements and loops. These are essential for decision-making and repetitive tasks.

If-Else Statements

# If-else example in R
score <- 85

if (score >= 50) {
    print("Pass")
} else {
    print("Fail")
}

For Loops

# Looping through numbers 1 to 5
for (i in 1:5) {
    print(i)
}

Functions in R

Functions in R allow us to reuse code efficiently. They take inputs, process them, and return results. 

# Defining a function in R
greet <- function(name) {
    return(paste("Hello,", name))
}

# Calling the function
print(greet("John"))

Data Visualization in R

R is widely used for data visualization. The ggplot2 package is one of the most popular tools for creating high-quality graphs. 

# Using ggplot2 for data visualization
library(ggplot2)

# Sample data
data(mtcars)

# Scatter plot of MPG vs Horsepower
ggplot(mtcars, aes(x=mpg, y=hp)) + geom_point()
